# Salary Prediction using Linear Regression

This project predicts employee salaries based on years of experience using a simple machine learning model.

## Steps
1. Load and clean the dataset (`Salary.csv`)
2. Split data into training and testing sets
3. Train a Linear Regression model
4. Predict salaries and visualize results

## Technologies Used
- Python
- pandas
- scikit-learn
- matplotlib

## How to Run
1. Install dependencies:  
   `pip install -r requirements.txt`
2. Run the notebook or script:  
   `python salary_prediction.py` or open `salary_prediction.ipynb`

## Output
- Graphs showing predicted vs actual salaries
